# dreamer-focus-app
# 🌙 Dreamer

> *An immersive productivity web app where focus meets the cosmos.*

🌐 **Live Demo:** https://dreamer-focus.netlify.app

---

## ✨ About

Dreamer transforms productivity into a calming cosmic experience. Instead of a traditional Pomodoro timer, Dreamer surrounds users with an interactive night-sky environment featuring animated moons, a cosmic companion, achievements, dream journaling, and beautiful visual effects.

Designed to make focus feel peaceful rather than stressful.

---

## 🌌 Features

- 🌙 Animated Dream Portal
- ⏳ Pomodoro Focus Timer
- 🐋 Cosmic Whale Companion
- 🌕 Dynamic Moon Phase Progression
- 📖 Dream Journal
- 🎯 Goal Tracker
- 🏆 Achievement System
- 📊 Productivity Statistics
- 🎨 Multiple Cosmic Themes
- ✨ Glassmorphism UI
- 🌌 Interactive Starfield & Particles
- 📱 Fully Responsive Design
- 💾 Local Storage Support
